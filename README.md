# Crypto Tracker

Crypto Tracker is an Android application that provides real-time cryptocurrency data, including prices, details, and trends of various coins. This project was developed with Jetpack Compose, following a modern approach to Android development.

## Features

- **Real-time Crypto Prices**: Displays current prices for various cryptocurrencies.
- **Detailed Coin Information**: Access detailed data and statistics for each coin.
- **Custom Price Change Graph**: Built from scratch to visually represent the price change of each cryptocurrency over time.
- **Dynamic Color Adaptation**: Adapts the theme based on user preferences for a more personalized experience.
- **Responsive UI**: Supports multiple screen sizes and orientations.
- **Dark Theme Support**: Provides light and dark modes.

## Technologies Used

- **Kotlin**
- **Jetpack Compose**
- **Koin**: For dependency injection 
- **ViewModel**: Manages UI-related data
- **Material Design**: Ensures a smooth and polished UI experience
